# 增强版存储统计信息 API 示例

## 修改内容

### 1. 时间戳类型修改
- **OldestTimestamp**: 从 `int64` 改为 `*time.Time` 类型
- **NewestTimestamp**: 从 `int64` 改为 `*time.Time` 类型
- 提供更友好的时间格式显示

### 2. 新增统计字段
- **AvgSeriesSize**: 平均每个序列占用数据大小（字节）
- **SamplesPerSecond**: 每秒存储样本数

### 3. 预计算字段
- **StorageSizeMB**: 存储大小（MB）
- **RetentionHours**: 数据保留时间（小时）

## 新的API响应格式

### 请求示例
```bash
curl -X GET "http://localhost:8000/system/storage/stats" \
  -H "Content-Type: application/json"
```

### 响应示例
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "stats": {
      "total_series": 49,
      "total_samples": 1310540,
      "storage_size": 4737250,
      "storage_size_mb": 4.517793655395508,
      "oldest_timestamp": "2025-08-05T10:00:00Z",
      "newest_timestamp": "2025-08-05T12:18:07.37Z",
      "retention_time": 8287,
      "retention_hours": 2.3019444444444446,
      "avg_series_size": 96678.57142857143,
      "samples_per_second": 158.12345678901234
    }
  }
}
```

## 字段详细说明

### 基础统计信息
| 字段名 | 类型 | 说明 | 示例值 |
|--------|------|------|--------|
| total_series | int64 | 总时间序列数量 | 49 |
| total_samples | int64 | 总样本数量 | 1310540 |
| storage_size | int64 | 存储大小（字节） | 4737250 |
| storage_size_mb | float64 | 存储大小（MB） | 4.52 |

### 时间信息
| 字段名 | 类型 | 说明 | 示例值 |
|--------|------|------|--------|
| oldest_timestamp | time.Time | 最老数据时间戳 | "2025-08-05T10:00:00Z" |
| newest_timestamp | time.Time | 最新数据时间戳 | "2025-08-05T12:18:07.37Z" |
| retention_time | int64 | 数据保留时间（秒） | 8287 |
| retention_hours | float64 | 数据保留时间（小时） | 2.30 |

### 新增统计信息
| 字段名 | 类型 | 说明 | 示例值 |
|--------|------|------|--------|
| avg_series_size | float64 | 平均每个序列占用数据大小（字节） | 96678.57 |
| samples_per_second | float64 | 每秒存储样本数 | 158.12 |

- **总存储**: 4.52 MB，存储量适中

### 采集性能
- **采集频率**: 0.31秒/次，高频采集
- **吞吐量**: 158样本/秒，中等负载
- **数据密度**: 26,745样本/序列，密度较高

## 监控告警建议

### 存储告警
- 存储大小超过100MB时告警
- 平均序列大小超过1MB时告警
- 每秒样本数超过1000时告警

### 性能告警
- 数据保留时间少于1小时时告警
- 采集频率超过1秒/次时告警
- 存储效率低于2字节/样本时告警
